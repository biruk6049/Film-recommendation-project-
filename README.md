🎬 Film Recommendation Website

A modern and responsive Movie Recommendation Web App built with HTML, CSS, and JavaScript that fetches movie data from TMDB API. Features include browsing Popular, Top Rated, and Now Playing movies, searching dynamically, a Favorites/Watchlist toggle, Dark/Light theme switch, responsive layout for all devices, movie details modal, IMDb rating badges, glass UI design, hover animations, loading skeleton, smooth animations, and a footer with social media icons. Technologies used: HTML5, CSS3, JavaScript (ES6), TMDB API, and LocalStorage. Project structure: index.html, style.css, script.js, api.js, README.md. Developed by Group 5 for educational purposes.
