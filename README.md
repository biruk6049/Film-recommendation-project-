🎬 Film Recommendation Website

A modern and responsive Movie Recommendation Web App built using HTML, CSS, and JavaScript.
This project fetches movie data from an API and displays popular, top-rated, and categorized films with an interactive UI.

🚀 Features
 Browse Popular Movies
 Top Rated Movies
 Now Playing Movies
 Search Movies
 Watchlist / Favorite Toggle
 Dark / Light Theme Toggle
 Fully Responsive (Mobile, Tablet, Desktop)
 Movie Details Modal
 IMDb Rating Badge
 Modern Glass UI Design

🛠 Technologies Used
HTML5
CSS3 (Glass UI + Responsive Design)
JavaScript (ES6)
Movie API (TMDB API)
LocalStorage (Favorites)

⚙️ How It Works
Fetch movies from Movie API
Display movies in responsive grid
Click movie → open details modal
Search movies dynamically
Toggle dark/light theme
Add movies to watchlist
Responsive UI for all devices
